## 영화관리 프로그램

### 팀원
|김해룡|염설화|🤴한슬기|
|---|---|---|

### 프로젝트 소개
> 등록된 영화들을 장르별로 조회하여 회원들이 직접 좋아요를 누르고, 좋아요 갯수를 바탕으로 영화를 추천해주는 프로그램

----------------------------------
### 주요 서비스
1. 사용자
    + 로그인 후 장르별로 영화 목록 조회 가능
    + 장르별로 좋아요 수가 많은 영화를 추천받음

2. 관리자
    + 영화 정보를 조회/삭제 가능
    + 영화 등록/수정 가능

---------------------------------
